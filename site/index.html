The problem

AI tools weren't built for developers like you.

You live in the terminal. You think in keystrokes. But every AI coding tool either locks you into a GUI, charges a subscription for basic features, or requires a 500 MB Electron app just to autocomplete a function.

What you're stuck with

  • $20/month subscriptions for basic chat
  • Electron apps eating 500 MB+ of RAM
  • Locked to a single AI provider
  • No offline mode -- cloud or nothing
  • Mouse-driven UIs that break your flow
  • Closed source with no way to extend

What Nerve gives you

  • Free and open source (MIT license)
  • 7.7 MB binary, no runtime dependencies
  • 6 providers -- switch with Ctrl+T
  • Ollama for fully offline, local AI
  • Vim keybindings and keyboard-first design
  • 166 built-in expert prompts

Agent Mode

Let Nerve do the work.

Type /agent on and describe what you need. Nerve reads files, writes code, runs commands, and verifies -- all autonomously. Every change is tracked with a git safety net.

nerve - agent
> Fix the auth bug and add tests
▶ read_file src/auth.rs
▶ search_code "fn login" src/
▶ edit_file src/auth.rs:45-52
▶ write_file tests/auth_test.rs
▶ run_command cargo test auth
✓ Fixed validation in login handler, added 3 tests. All passing.
/agent undo to rollback  |  /agent diff to review

Get started

Up and running in 30 seconds.

Download binary

# Linux
curl -LO https://github.com/Artaeon/nerve/releases\
/latest/download/nerve-linux-x86_64.tar.gz
tar xzf nerve-linux-x86_64.tar.gz
mv nerve ~/.local/bin/

# macOS (Apple Silicon)
curl -LO https://github.com/Artaeon/nerve/releases\
/latest/download/nerve-macos-arm64.tar.gz
tar xzf nerve-macos-arm64.tar.gz
mv nerve /usr/local/bin/

Build from source

# Clone and build
git clone https://github.com/Artaeon/nerve.git
cd nerve
cargo build --release

# Install
cp target/release/nerve ~/.local/bin/

# Verify
nerve --help

Requires Rust 1.85+ (edition 2024).
